AI CV upload for PDF/DOCX/image needs backend.

1) Open backend/.env
2) Paste your key:
   OPENAI_API_KEY=sk-proj-...
3) Save the file
4) Double click start-backend.bat
5) Keep that backend window open
6) Open frontend/index.html again

The backend now also reads OPENAI_API_KEY from project-root .env if you placed it there.
TXT upload works locally without backend.

AUTH UPDATE:
- Register now saves the user email and a secure password hash in backend/data/users.json.
- Login checks the saved password. Wrong password cannot enter.
- Auto Apply uses the saved/login email as CC/reply-to and rejects unregistered emails.
